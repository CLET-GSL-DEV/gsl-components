# NetworkOperator

Network operator selector with operator image thumbnails (MTN, Vodafone, AirtelTigo, Glo), search filtering, and keyboard navigation. Wraps a Radix popover with a trigger that shows the selected operator's logo and name.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected operator value |
| `defaultValue` | `string` | — | Initial value for uncontrolled mode |
| `onChange` | `(value: string \| null) => void` | — | Selection handler |
| `options` | `NetworkOperatorOption[]` | Default list (MTN, Vodafone, AirtelTigo, Glo) | Custom operator list |
| `placeholder` | `string` | `"Select operator"` | Trigger placeholder |
| `invalid` | `boolean` | `false` | Show error border styling |
| `disabled` | `boolean` | `false` | Disable interaction |
| `classNames` | `NetworkOperatorClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the root element |

Also accepts remaining `HTMLAttributes<HTMLDivElement>`, excluding `onChange`.

## Usage with react-hook-form

```tsx
import { useForm } from "react-hook-form";
import { Form, FormField, Field, FieldControl, FieldLabel, FieldError, NetworkOperator } from "@rfdtech/components";

function OperatorForm() {
  const form = useForm<{ operator: string }>({ defaultValues: { operator: "mtn" } });

  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit((d) => console.log(d.operator))}>
        <FormField
          control={form.control}
          name="operator"
          rules={{ required: "Select an operator" }}
          render={({ field, fieldState }) => (
            <Field invalid={!!fieldState.error}>
              <FieldLabel>Network operator</FieldLabel>
              <FieldControl>
                <NetworkOperator value={field.value} onChange={field.onChange} />
              </FieldControl>
              <FieldError>{fieldState.error?.message}</FieldError>
            </Field>
          )}
        />
      </form>
    </Form>
  );
}
```

## Notes

- Default operators: MTN, Vodafone, AirtelTigo, Glo — with logo image URLs.
- Pass your own `options` array for a custom list.
- Uncontrolled mode: use `defaultValue` to set the initial operator. Without it, no operator is pre-selected.
- Search filters by operator name (case-insensitive).
- Images render as rounded thumbnails. Falls back to the first letter of the label if no image is provided or if the image fails to load.
