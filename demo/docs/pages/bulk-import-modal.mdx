# BulkImportModal

Four-step modal wizard for importing spreadsheet data (`.xlsx`, `.xls`, `.csv`). Parsing and validation run entirely in the browser.

## Example

<ExampleTabs
  title="Open import wizard"
  preview={<BulkImportPreview />}
  code={bulkImportSource}
/>

Click **Open bulk import** to launch the multi-step import modal with sample data.

## Steps

1. **Upload Document** — preview expected columns, then upload `.xlsx`, `.xls`, or `.csv`
2. **Select header row** — choose the header row with radio buttons
3. **Match Columns** — map each uploaded column to a target field
4. **Validate data** — review rows, discard invalid rows, then **Confirm**

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | — | Open state callback |
| `fields` | `BulkImportField[]` | — | Target field schema for column matching |
| `onComplete` | `(result: BulkImportResult) => void` | — | Called with mapped rows on import |
| `title` | `string` | `Bulk import` | Modal title |
| `maxFileSizeBytes` | `number` | `5242880` | Maximum upload size (5 MB) |
| `allowImportWithWarnings` | `boolean` | `false` | Allow import when only warnings exist |
| `defaultState` | `BulkImportFlowDefaultState` | — | Seed initial flow state for progress preservation across open/close cycles |

## Notes

- Also exported: `useBulkImportFlow`, step components, and spreadsheet utilities.
- Discarded rows are excluded from `onComplete`.
- The modal uses `useConfirmBeforeUnload` to warn on page refresh/close when a file has been uploaded but not confirmed.
- The example uses `useModalSearchParam("bulk-import")` for URL-driven open state (`?modal=bulk-import`). See [Modal URL state](/docs/modal#url-state).
