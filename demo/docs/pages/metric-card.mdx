# MetricCard

Metric display card for dashboards and overviews. Renders a compact card with a label, large value, optional trend arrow with percentage, icon, and description.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Metric label shown above the value (uppercased via CSS) |
| `value` | `string \| number` | — | The primary metric value |
| `icon` | `ReactNode` | — | Optional icon rendered next to the label (always muted tint) |
| `description` | `string` | — | Subtitle shown below the value row |
| `trend` | `"up" \| "down" \| "neutral"` | — | Trend direction; renders an arrow icon |
| `trendValue` | `string` | — | Formatted trend text (e.g. `"+12.5%"`) |
| `variant` | `"default" \| "outline"` | `"default"` | Card style — `"outline"` is a bordered, no-fill card with chevron trend icons |
| `color` | `string` | — | Custom CSS color for the trend indicator; overrides variant |
| `classNames` | `MetricCardClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the root element |

Also accepts remaining `HTMLAttributes<HTMLDivElement>`.

## Variants

Variants only affect the **trend indicator color**, not the background or text of the card:

| Variant | Trend color |
|---------|-------------|
| `"default"` | `--gsl-success` (up), `--gsl-error` (down), `--gsl-text-muted` (neutral) |
| `"primary"` | `--gsl-primary` |
| `"success"` | `--gsl-success` |
| `"warning"` | `--gsl-warning` |
| `"error"` | `--gsl-error` |

Override with the `color` prop (any CSS value) for custom accent colors.

### Outline variant

Set `variant="outline"` for a bordered, no-fill card. It differs structurally from the default:

- **No background** — transparent card with a `1px` border in `--gsl-border`, radius `--gsl-radius-base`.
- **Chevron trend icons** — `up` renders a chevron up, `down` a chevron down, `neutral` a dash.
- **No `+`/`-` prefix** — a leading sign is stripped from `trendValue` (pass `"12.5%"`, not `"+12.5%"`).
- **Larger value** — `32px` font size.
- **Padding `24px`** with a unified `8px` gap between the label, value, and description.

```tsx
<MetricCard
  variant="outline"
  label="Revenue"
  value="$128.4k"
  description="Total revenue this quarter"
  trend="up"
  trendValue="12.5%"
/>
```

## Types

### `MetricCardProps`

See the props table above.

### `MetricCardClassNames`

| Key | Applied to |
|-----|------------|
| `root` | Root element (`.gsl-metric-card`) |
| `icon` | Icon wrapper (`.gsl-metric-card__icon`) |
| `label` | Label text (`.gsl-metric-card__label`) |
| `value` | Value text (`.gsl-metric-card__value`) |
| `description` | Description text (`.gsl-metric-card__description`) |
| `trend` | Trend indicator (`.gsl-metric-card__trend`) |

Also exported: `MetricCardVariant`, `MetricTrend`.

## Design tokens

| Token | Default | Used for |
|-------|---------|----------|
| `--gsl-metric-card-value-size` | `28px` (`32px` outline) | Main value (header) font size |
| `--gsl-metric-card-value-color` | `--gsl-text` | Main value (header) color |
| `--gsl-metric-card-description-size` | `13px` | Description font size |

Override card background via `--gsl-default-metric-card-bg`, radius via `--gsl-metric-card-radius`, or shadow via `--gsl-metric-card-shadow`.

## Notes

- Background and text remain neutral across all variants — only the trend indicator changes color. Use the `color` prop for full control.
- The icon always renders in `--gsl-text-muted` regardless of variant.
- Card width is determined by the parent container. Use a grid or flex wrapper for layout.
- On mobile (`<= 768px`), padding reduces to 16px and value font size to 24px.
