# DateSelector

Date picker for choosing a single date from a calendar grid. Built on `@radix-ui/react-popover` with GSL styling, month/year navigation arrows, weekday headers, and native `Date.toLocaleDateString` formatting.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `Date \| null` | — | Controlled selected date |
| `defaultValue` | `Date \| null` | — | Default date (uncontrolled) |
| `onChange` | `(date: Date \| null) => void` | — | Selection handler |
| `min` | `Date` | — | Minimum selectable date (inclusive) |
| `max` | `Date` | — | Maximum selectable date (inclusive) |
| `formatOptions` | `Intl.DateTimeFormatOptions` | — | Options for `Date.toLocaleDateString()` formatting |
| `placeholder` | `string` | `"Select date"` | Trigger text when no date is selected |
| `invalid` | `boolean` | `false` | Show error border styling |
| `disabled` | `boolean` | `false` | Disable interaction |
| `name` | `string` | — | Form field name (for react-hook-form) |
| `onBlur` | `() => void` | — | Called when the trigger loses focus |
| `classNames` | `DateSelectorClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the root element |

Also accepts remaining `HTMLAttributes<HTMLDivElement>`.

## Usage with react-hook-form

```tsx
import { useForm } from "react-hook-form";
import { Form, FormField, Field, FieldControl, FieldLabel, FieldError, DateSelector } from "@rfdtech/components";

function DateForm() {
  const form = useForm<{ startDate: Date | null }>({ defaultValues: { startDate: null } });

  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit((d) => console.log(d.startDate))}>
        <FormField
          control={form.control}
          name="startDate"
          rules={{ required: "Start date is required" }}
          render={({ field, fieldState }) => (
            <Field invalid={!!fieldState.error}>
              <FieldLabel>Start date</FieldLabel>
              <FieldControl>
                <DateSelector value={field.value} onChange={field.onChange} />
              </FieldControl>
              <FieldError>{fieldState.error?.message}</FieldError>
            </Field>
          )}
        />
      </form>
    </Form>
  );
}
```

## Notes

- Formatting uses `Date.toLocaleDateString()` — locale-aware and does not require a date library.
- Navigation arrows move by month. Clicking the month/year text resets to today.
- Days from adjacent months are shown muted and selectable.
- The popover calendar uses `--gsl-shadow-md` and matches the trigger width.

## Don't

- Override borders with `!border-0`/`!shadow-none` style class overrides — let it keep its default border, like Dropdown.
