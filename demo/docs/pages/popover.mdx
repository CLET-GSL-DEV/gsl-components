# Popover

Compound popover primitives for floating panels anchored to a trigger. Wraps `@radix-ui/react-popover`; `PopoverContent` applies default GSL surface styles.

## Components

### `Popover`

Root wrapper. Accepts Radix `Popover.Root` props such as `open`, `defaultOpen`, and `onOpenChange`.

### `PopoverTrigger`

Wraps the element that opens the popover. Use `asChild` to merge props onto a child such as `Button`.

### `PopoverContent`

Styled floating panel. Accepts Radix content props (`side`, `align`, `sideOffset`, etc.) plus GSL `classNames`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `PopoverContentClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the content element |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Preferred placement |
| `align` | `"start" \| "center" \| "end"` | `"center"` | Alignment relative to trigger |
| `sideOffset` | `number` | `0` | Distance from the trigger |

Also accepts remaining Radix `Popover.Content` props.

### Other exports

| Export | Radix primitive | Notes |
|--------|-----------------|-------|
| `PopoverPortal` | `Portal` | Renders content in a portal |
| `PopoverAnchor` | `Anchor` | Alternative positioning anchor |
| `PopoverClose` | `Close` | Closes the popover when activated |

## Types

### `PopoverContentProps`

Extends Radix `PopoverContentProps` with optional `classNames`.

### `PopoverContentClassNames`

| Key | Applied to |
|-----|------------|
| `content` | Content element (`.gsl-popover`) |

## Design tokens

| Token | Default | Used for |
|-------|---------|----------|
| `--gsl-popover-font` | `var(--gsl-font-body)` | Font family override |

## Notes

- Portaled content inherits the active theme via `data-gsl-theme` on `document.documentElement` when using `ThemeProvider`.
- Use `PopoverTrigger asChild` so focus and keyboard behavior stay on your button or link.
- Wrap menu actions in `PopoverClose asChild` so the panel dismisses after selection.
- Use `gsl-popover--menu`, `gsl-popover__menu`, and `gsl-popover__menu-item` for compact action menus inside `PopoverContent`.
- Base `gsl-popover` styles are always applied; `classNames.content` and `className` append consumer classes.
