# Sheet

Compound sheet primitives for slide-out panels anchored to a viewport edge. Built on Radix Dialog with GSL styling. Use for filters, settings, detail panels, and other secondary workflows.

| Component | Best for |
|-----------|----------|
| [**Dialog**](/docs/dialog) | Small centered modals |
| [**Modal**](/docs/modal) | Near full-viewport workflows |
| **Sheet** | Edge-sliding panels (`top`, `right`, `bottom`, `left`) |
| [**Sidebar**](/docs/sidebar) | Persistent app navigation rail |

## Components

### `Sheet`

Root wrapper. Accepts Radix `Dialog.Root` props such as `open`, `defaultOpen`, and `onOpenChange`.

### `SheetTrigger`

Wraps the element that opens the sheet. Use `asChild` to merge props onto a child such as `Button`.

### `SheetPortal`

Renders overlay and content in a portal. Wrap `SheetOverlay` and `SheetContent` inside the portal.

### `SheetOverlay`

Styled backdrop behind the sheet panel.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `SheetOverlayClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the overlay element |

Also accepts remaining Radix `Dialog.Overlay` props.

### `SheetContent`

Sliding panel shell.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `SheetSide` | `"right"` | Edge the sheet slides from (`top`, `right`, `bottom`, `left`) |
| `showCloseButton` | `boolean` | `false` | Shows a top-right close control with an accessible label |
| `classNames` | `SheetContentClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the panel element |

Also accepts remaining Radix `Dialog.Content` props.

### `SheetHeader`

Top chrome bar for titles and supporting text.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `SheetHeaderClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the header element |

### `SheetTitle`

Accessible sheet heading. Render inside `SheetHeader`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `SheetTitleClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the title element |

### `SheetDescription`

Optional supporting text below the title.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `SheetDescriptionClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the description element |

### `SheetBody`

Scrollable main content area.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `SheetBodyClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the body element |

### `SheetFooter`

Bottom action bar. Defaults to end-aligned flex layout for buttons.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `classNames` | `SheetFooterClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the footer element |

### Other exports

| Export | Radix primitive | Notes |
|--------|-----------------|-------|
| `SheetClose` | `Close` | Closes the sheet when activated |

## Types

### `SheetSide`

`"top" | "right" | "bottom" | "left"`

### `SheetOverlayProps` / `SheetContentProps` / `SheetHeaderProps`

Extend the corresponding element or Radix props with optional `classNames`.

### `SheetTitleProps` / `SheetDescriptionProps`

Extend Radix title and description props with optional `classNames`.

### `SheetBodyProps` / `SheetFooterProps`

Extend native `div` / `footer` attributes with optional `classNames`.

### `SheetOverlayClassNames`

| Key | Applied to |
|-----|------------|
| `overlay` | Overlay element (`.gsl-sheet__overlay`) |

### `SheetContentClassNames`

| Key | Applied to |
|-----|------------|
| `content` | Panel element (`.gsl-sheet`) |
| `close` | Built-in close button (`.gsl-sheet__close`) |

### `SheetHeaderClassNames` / `SheetTitleClassNames` / `SheetDescriptionClassNames`

| Type | Key | Applied to |
|------|-----|------------|
| `SheetHeaderClassNames` | `header` | Header element (`.gsl-sheet__header`) |
| `SheetTitleClassNames` | `title` | Title element (`.gsl-sheet__title`) |
| `SheetDescriptionClassNames` | `description` | Description element (`.gsl-sheet__description`) |

### `SheetBodyClassNames` / `SheetFooterClassNames`

| Type | Key | Applied to |
|------|-----|------------|
| `SheetBodyClassNames` | `body` | Body element (`.gsl-sheet__body`) |
| `SheetFooterClassNames` | `footer` | Footer element (`.gsl-sheet__footer`) |

## Notes

- Portaled content inherits the active theme via `data-gsl-theme` on `document.documentElement` when using `ThemeProvider`.
- Left and right sheets use `width: min(400px, 100vw)` and full viewport height. Top and bottom sheets use full width and `height: min(50dvh, 100dvh)`.
- `SheetBody` scrolls internally when content exceeds the available height.
- The built-in close control uses a 40×40px hit area for touch-friendly dismissal.
- Slide animations are disabled when `prefers-reduced-motion: reduce` is set.
- Base `gsl-sheet*` classes are always applied; `classNames` and `className` append consumer classes for Tailwind utilities.
