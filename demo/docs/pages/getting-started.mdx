# Getting started

`@rfdtech/components` is a shared React library for Ghana School of Law projects.

## Install

```bash title="terminal"
npm install @rfdtech/components
```

Requires React 18+. npm 7+ auto-installs `react` and `react-dom` as peer dependencies. Radix UI and other runtime packages are included as dependencies of `@rfdtech/components`.

## Styles

Component styles load automatically when you import from `@rfdtech/components`. For reliable styling in production, add this once in your app entry (`main.tsx` or `App.tsx`):

```ts title="main.tsx"
import "@rfdtech/components/style.css";
```

## Theme

Wrap your app in `ThemeProvider` so all components share light, dark, or system themes:

```tsx title="App.tsx"
import { ThemeProvider } from "@rfdtech/components";

<ThemeProvider defaultTheme="system">
  <App />
</ThemeProvider>
```

See the [Theme](/docs/theme) page for `useTheme()`, token reference, and controlled mode.

## Basic usage

```tsx title="App.tsx"
import { BulkImportModal, ThemeProvider } from "@rfdtech/components";
import "@rfdtech/components/style.css";

<ThemeProvider defaultTheme="system">
  <BulkImportModal
    open={open}
    onOpenChange={setOpen}
    title="Import students"
    fields={fields}
    onComplete={(result) => console.log(result.rows)}
  />
</ThemeProvider>
```

## Requirements

- React 18+
- A bundler that processes CSS (Vite, Webpack, etc.)

## Theming

All components share design tokens from `theme.css`. Use `ThemeProvider` for light, dark, and system modes — see the [Theme](/docs/theme) page.

## Tips

- After upgrading the package, clear Vite's dependency cache if styles look stale: `rm -rf node_modules/.vite`
- Browse the component pages in this sidebar for previews, code examples, and props reference
- Try the interactive [demo](/) for combined examples
