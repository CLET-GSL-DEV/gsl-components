# Card

Surface card wrapper for grouping content. Uses `--gsl-surface-card` for background and `--gsl-card-padding` for spacing — both overridable via CSS custom properties or `className`.

## Example

<ExampleTabs title="Card variants" preview={<CardPreview />} code={cardSource} />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Card content |

Also accepts remaining `HTMLAttributes<HTMLDivElement>`.

## Design tokens

| Token | Fallback | Used for |
|-------|----------|----------|
| `--gsl-surface-card` | `--gsl-bg` | Card background |
| `--gsl-card-padding` | `20px` | Inner padding (16px on mobile) |

Override padding via `--gsl-card-padding` or tailwind padding classes through `className`.

## Notes

- On mobile (`<= 768px`), padding reduces to 16px via `--gsl-card-padding`.
- Use `className` to override background, border-radius, shadow, or any root style.
