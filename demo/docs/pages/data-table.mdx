# DataTable

Read-focused data table with column sorting and pagination. Pass `data` for client-side behavior; use `loading` when data is fetched by the parent.

Composes the standalone [Pagination](/docs/pagination) component for its footer.

## Example

<ExampleTabs title="Staff directory" preview={<DataTablePreview />}>

```tsx
import { DataTable } from "@rfdtech/components";

const { data, loading } = useStaff();

<DataTable
  columns={columns}
  data={data}
  loading={loading}
  emptyText="No staff found."
  pageSize={10}
  onRowClick={(row) => console.log(row.name)}
/>
```

</ExampleTabs>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `columns` | `DataTableColumn[]` | — | Column definitions |
| `data` | `T[]` | — | Rows to display |
| `loading` | `boolean` | `false` | Show spinner and disable controls |
| `pageSize` | `number` | `10` | Initial rows per page |
| `pageSizeOptions` | `number[]` | `[10, 25, 50]` | Page size choices |
| `emptyText` | `string` | `No results found.` | Empty state message |
| `loadingLabel` | `string` | `Loading data` | Accessible label for loading spinner |
| `onRowClick` | `(row: T) => void` | — | Row click handler |
