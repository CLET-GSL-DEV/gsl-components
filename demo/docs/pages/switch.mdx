# Switch

Binary on/off toggle built on Radix `Switch`. Use it for settings that take effect immediately, unlike `Checkbox`, which is typically used inside a form that requires an explicit submit.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | — | Uncontrolled initial checked state |
| `onCheckedChange` | `(checked: boolean) => void` | — | Fires when the switch is toggled |
| `label` | `ReactNode` | — | Renders a "Switch + Text" variant |
| `labelPosition` | `"left" \| "right"` | `"right"` | Side the label renders on when `label` is set |
| `disabled` | `boolean` | `false` | Disables interaction |
| `invalid` | `boolean` | `false` | Error styling + `aria-invalid` |
| `required` | `boolean` | — | Marks the field required for form submission |
| `name` | `string` | — | Form field name |
| `value` | `string` | — | Form value when checked |
| `id` | `string` | — | Overrides the auto-generated id |
| `aria-label` | `string` | — | Accessible name when no visible `label` is set |
| `classNames` | `SwitchClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the root (same as `classNames.root`) |

## Types

### `SwitchProps`

Props for `Switch`. Not `forwardRef` — matches the `Checkbox` convention of a plain function component with an internal Radix `Switch.Root`/`Switch.Thumb` pair.

### `SwitchClassNames`

Part-level class names merged after base `clet-switch*` styles:

| Key | Applied to |
|-----|------------|
| `root` | Outer wrapper (`<label>` when `label` is set, else `<span>`) |
| `track` | The Radix `Switch.Root` track button |
| `thumb` | The Radix `Switch.Thumb` |
| `label` | The label `<span>` |

## Design tokens

| Token | Default | Used for |
|-------|---------|----------|
| `--clet-switch-width` | `44px` | Track width |
| `--clet-switch-height` | `24px` | Track height |
| `--clet-switch-thumb-size` | `20px` | Thumb diameter |
| `--clet-switch-padding` | `2px` | Inset between track edge and thumb |
| `--clet-switch-font` | `var(--clet-font-body)` | Font family for the label |
| `--clet-switch-label-size` | `var(--clet-text-size-base)` | Label font size |

## Notes

- Checked track uses `--clet-primary`; unchecked uses `--clet-border-strong`. No custom blue accent is used — the design's on-brand navy matches the library's existing primary token.
- Hover, focus (`--clet-focus` outline), and disabled (`opacity: 0.6`) states are handled purely in CSS via Radix's `data-state`/`data-disabled` attributes.
- Clicking the visible `label` toggles the switch, same as `Checkbox`.
