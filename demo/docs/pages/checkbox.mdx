## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | — | Initial checked state (uncontrolled) |
| `onCheckedChange` | `(checked: boolean) => void` | — | Called when checked state changes |
| `label` | `ReactNode` | — | Visible label beside the control |
| `disabled` | `boolean` | `false` | Disables the checkbox |
| `required` | `boolean` | — | Marks the field as required in forms |
| `name` | `string` | — | Form field name |
| `value` | `string` | — | Form field value when checked |
| `id` | `string` | — | Control id (auto-generated if omitted) |
| `aria-label` | `string` | — | Accessible name when `label` is omitted |
| `classNames` | `CheckboxClassNames` | — | Part-level class overrides |
| `className` | `string` | — | Merged onto the root wrapper |

## Types

### `CheckboxProps`

Props for `Checkbox`. Supports controlled and uncontrolled usage with an optional visible label.

### `CheckboxClassNames`

Part-level class names merged after base `gsl-checkbox*` styles:

| Key | Applied to |
|-----|------------|
| `root` | Outer wrapper (`.gsl-checkbox`) |
| `control` | Checkbox control (`.gsl-checkbox__control`) |
| `indicator` | Checked indicator container |
| `icon` | Lucide check icon |
| `label` | Visible label text |

## Design tokens

| Token | Default | Used for |
|-------|---------|----------|
| `--gsl-checkbox-font` | `var(--gsl-font-body)` | Font family override |

## Notes

- Provide `label` or `aria-label` so the checkbox has an accessible name.
- Checked state shows a Lucide `Check` icon on the primary background.
- Base `gsl-*` classes are always applied; `classNames` and `className` append consumer classes for Tailwind utilities.
